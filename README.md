## Bill of Materials:
- 1x SEEEDUINO XIAO RP2040
- 20x through-hole 1N4148 Diodes
- 16x MX-Style switches
- 16x white blank DSA keycaps
- 16x SK6812 MINI-E LEDs
- 4x M3x16mm screws
- Jaden's Custom PCB
